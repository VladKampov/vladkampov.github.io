Привет! Я очередной бот для Telegram. Но я сделан, чтобы жизнь КПИшников стала немножечко проще.

Пока что я умею только показывать расписание (но зато как!), но это пока что. Мой функционал неограничен и будет дополняться моими создателями в будущем. А если у тебя, мой друг, есть идеи о том, как можно меня усовершенствовать, то отправляй свои идеи вот сюда — kpibot@kampov.com

Чтобы я начал тебе помогать нужно всего лишь пройти по ссылке или найти меня по нику — @kpibot

 

Сейчас я уже умею следующее:

  • /setgroup [Номер твой группы] — таким образом я тебя запомню для дальнейшей нашей совместной жизни.
    (Пример: /setgroup КВ-32 или /setgroup kv-32)
  • /today — этой командой я покажу тебе расписание на сегодня.
  • /teacher — вместе с названиями предметов покажу тебе имена преподавателей.
  • /tomorrow — этой командой я смогу тебе показать расписание на завтра
  • /next — расскажет тебе, какая же пара следующая
  • /where — покажет, где же ты вообще находишься (или должен находится)
  • /who — напомнит тебе, как зовут преподавателя, который стоит перед тобой
  • /tt — отображает полное расписание на все недели

 

А ещё к каждой из комманд можно добавить параметры

  • Номер группы в виде КВ-32 или kv-32 чтобы отобразить расписание для любой конкретной группы
    (Например, "/today kv-52" покажет расписание на сегодня для группы КВ-52)
  • Номер недели в ввиде w1 и w2 для вывода результата для первой или второй недели обучения соответственно
    (Например, "/tt w1" покажет полное расписение первой недели, а "/today w2" посмотрит, какой сегодня день недели, и покажет расписание этого дня на второй неделе)
  • Ты можешь посмотреть расписание на конкретный день используя "/today 1", где "1" — номер дня недели. Да, "/today mon" и "/today пн" работают тоже.

А самое главное, что все эти параметры ты можешь комбинировать! Причем в любом порядке!

Пару примеров:

  • /tt ia-25 w1
  • /tt w2 mon IA-52
  • /tt w2 mon IA-52
  • /tt KM-21 3 w1

Использование неправильных параметров выдаст ошибку
